# 📊 Madhav E-Commerce Sales Dashboard

## 🔹 Overview

This project presents an interactive sales dashboard built using **Microsoft Power BI** to analyze e-commerce data and generate actionable business insights. It enables users to explore sales performance, customer behavior, and product trends through dynamic visualizations.

---

## 🎯 Objectives

* Analyze overall **sales and profit performance**
* Identify **top-performing states and customers**
* Understand **category and sub-category contributions**
* Track **monthly profit trends**
* Analyze **customer payment preferences**

---

## 📈 Key Insights

* **Maharashtra** generates the highest sales
* **Clothing** category contributes the largest share
* **Cash on Delivery (COD)** is the most preferred payment method
* **January** records the highest profit
* **Printers** and **Phones** are the top profit-generating sub-categories

---

## ⚙️ Features

* KPI cards for:

  * Total Sales
  * Quantity
  * Profit
  * Average Order Value
* **State-wise sales analysis** (Bar Chart)
* **Category & payment mode distribution** (Donut Charts)
* **Monthly profit trends** (Column Chart)
* **Customer-wise and sub-category analysis**
* **Interactive filters** (Quarter selection)


---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use the interactive filters to explore insights
